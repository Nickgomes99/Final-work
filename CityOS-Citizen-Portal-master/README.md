# CityOS Citizen Portal

A modern web portal for citizens to access city services, view announcements, and interact with their local government.

## Features

- **Dashboard**: Overview of city services and personal information
- **Service Requests**: Submit and track service requests (potholes, street lights, etc.)
- **Bill Payments**: View and pay utility bills
- **Announcements**: Stay updated with city news and events
- **Contact**: Reach out to city departments

## Technologies

- HTML5
- CSS3
- JavaScript (Vanilla)
- Responsive Design

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. Explore the portal features
